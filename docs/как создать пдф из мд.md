## Как сделать PDF

Если запускаете `pandoc` из корня репозитория, добавьте `--resource-path`,
иначе картинки из `docs/runbooks/images/` не найдутся.

Надёжный вариант без LaTeX PDF-зависимостей:

```bash
pandoc --resource-path=docs/runbooks \
    --standalone --embed-resources \
    docs/runbooks/operator-iso-installer.md \
    -o operator-iso-installer.html
```

Откройте `operator-iso-installer.html` в браузере и выберите
`Print -> Save to PDF`.

Прямой PDF через pandoc требует полноценный LaTeX и конвертер SVG:

```bash
sudo apt install pandoc texlive-xetex texlive-latex-recommended texlive-lang-cyrillic librsvg2-bin

pandoc --resource-path=docs/runbooks \
    --pdf-engine=xelatex \
    -V mainfont="DejaVu Sans" \
    -V monofont="DejaVu Sans Mono" \
    docs/runbooks/operator-iso-installer.md \
    -o operator-iso-installer.pdf
```

Если `DejaVu Sans` отсутствует, проверьте доступные шрифты командой
`fc-list :lang=ru family` и подставьте любой шрифт с кириллицей, например
`Noto Sans` или `Liberation Sans`.
